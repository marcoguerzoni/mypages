Come lavorare

Giovanni, immagina di lavorare con GitLab. Funziona come GitHub, lo hai mai usato?

Di fatto, ogni progetto RStudio si sincronizza con il nostro account GitLab.

Possiamo fare commit e pull, oppure semplicemente copiare i nostri file R e lanciarli.

Ovviamente non è efficiente fare tutte le prove direttamente da lì.

Container e RStudio Server

L'ultimo layer di complessità è dato dal fatto che RStudio Server gira in un container.

Appena Giovanni carica RStudio, ci manda un esempio.

Proviamo l'esempio e gli spieghiamo cosa non abbiamo capito.

Runtime e Avvio

Comandi utili:

runtime cli nome_programma path ed eventuali opzioni
runtime.sh (fa partire RStudio Server)

Oppure:

rstudio
parte rstudio server

Directory corretta

Devi essere nella directory exec dove si trovano gli script aggiuntivi per i package R.
